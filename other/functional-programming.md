# 函数式编程

## 函数式编程是什么

- 一种编程范式
- 函数式编程中的函数指的是一种映射关系
- 用来描述数据（函数）之间的映射

## 特性

- 函数是一等公民
  - 函数与其他数据类型一样
  - 函数可作为返回结果
  - 函数可作为参数
- 高阶函数
  - 将一个或多个函数作为参数传入
  - 返回一个函数作为结果
- 闭包
  - 缓存当前上下文执行环境的词法作用域
- 纯函数
  - 没有副作用
  - 纯函数优点：可降低复杂性、其他特性（如可缓存性）
- 柯里化
  - 给函数传递一部分参数，返回一个函数，处理剩余参数
- 函数组合
  - 纯函数及柯里化会写出不优雅的洋葱代码，为解决函数嵌套，需要用到‘函数组合’
- PointFree
  - 把一些对象自带的方法转化成纯函数
- 函子Functor
  - 遵守一些特定规则的容器类型
  - 是一个对于函数调用的抽象，我们赋予容器去调用函数的能力。
